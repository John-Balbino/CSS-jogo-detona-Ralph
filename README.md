Projeto Dio: Detona Ralph jogo

Este projeto trata-se de jogo do detona ralph  do modelo acerte o alvo, este mesmo alvo fica se movendo de forma aleatorio dentro
dos limites estabelicidos, o jogo conta com um sistema de pontuação, timer e vidas !

o intuito deste projeto é desenvolver melhor a logica de programção e como utilizar o JS para manipular a DOM !
